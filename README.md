# ✈️ Airline Reservation System (C++ OOP)

## 📌 Project Overview
The Airline Reservation System is a console-based application developed in C++ using Object-Oriented Programming (OOP) principles. It simulates real-world airline booking operations such as searching flights, booking seats, generating tickets, and cancelling reservations.

The system is designed to demonstrate modular programming, encapsulation, and real-world problem solving using C++ classes and objects.

---

## 🚀 Features

- 🔍 Search flights by source and destination
- ✈️ View all available flights
- 🎟️ Book flight tickets with automatic seat allocation
- ❌ Cancel booked tickets using Ticket ID
- 🧾 Generate and display ticket details
- 🔠 Case-insensitive input handling (Lahore = lahore = LAHORE)
- 🪑 Automatic seat management system
- 🌍 Supports domestic and international flight routes

---

## 🧠 OOP Concepts Used

- **Encapsulation** → Data hidden inside classes (Flight, Ticket)
- **Abstraction** → User interacts only with system menu
- **Modularity** → Separate classes for different responsibilities
- **Data Management** → Seat tracking and ticket system
